Style Guide

Welcome to The Pudding style guide. Use this as a reference to structure your HTML and CSS, and follow our best practices.

Usage

Everything you need is already included in your new project template. Simply follow these guidelines and use the proper CSS classes outlined and you should be good to go.

Typography

Fonts

Fonts are loaded async and use the FOUT practice. We have three font families:

  • Canela: (class name: tk-canela)
  • Publico: (class name: tk-publico)
  • Atlas Grotesk: (class name: tk-atlas)

Simply include the class on the element, and all children will inherit it. Publico is included on the body tag by default. 

<p class='tk-atlas'>Test</p>

Use the font-weight CSS property to set the weight.

  • Canela: 300, 700
  • Publico: 400, 700
  • Atlas: 400, 500, 600

Font size classes

Use the following convenience classes to size your text. They apply both font-size and line-height, so you may have to tweak depending on your needs.

.xxx-small (10px)

.xx-small (11px)

.x-small (12px)

.small (14px)

.root (16px)

.medium (17px - default)

.large (24px)

.x-large (32px)

.xx-large (40px)

.xxx-large (48px)

.xxxx-large (64px)
